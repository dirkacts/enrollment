Definitions

Dashboard - A Dashboard is a single page visualization that is comprised of one or more charts or tables. This workbook contains three Dashboards: Current Enrollment Analysis, Weekly Enrollment Analysis, and Enrollment Analysis by Year. Each Dashboard is represented by a tab at the top of the Workbook.

Workbook - A Workbook is a group of one or more Dashboards. The current Workbook is entitled "Enrollment Visualization.""

Enrollment - Enrollment is the total number of students officially on record at the specified point(s) in time.

Enrollment (Filtered) - The enrollment after the application of any Action Filters and Quick Filters (see "Filters").

Total Enrollment - The enrollment after the application of any Quick Filters. Total Enrollment is not affected by Action Filters (see "Filters").

Grade Demographics - The balance in the population, by grade, after the "Grade By:" Parameter Filter has been selected (see "Filters"). Grade demographics can be viewed by ethnicity, gender, IEP Status, Section 504 status, LEP status, Service Type, or State Excluded students. These demographics will be defined in the "Filters" section.

Tab Layout and Navigation

Information in Tableau is presented in tabs, or Dashboards, which are similar to sheets in Microsoft Excel. Named tabs contain separate Dashboards with different information allowing users to switch between them. To switch between Dashboards, click on the Dashboard title at the top.

Dashboard Format

All Dashboards are designed using the same basic format. The main area features the visualization and includes Action Filters (discussed below). The right column contains Quick Filters, Parameter Filters (both are discussed below) and graph legends.

Tooltips

Tooltips are displayed when hovering over a graph element of a chart. The tooltip will show what is being measured and the enrollment for that segment of the population.

Filters

Filters allow the user to look at the enrollment for a specific segment of the population. There are three types of filters available in the Enrollment Visualization workbook.

Action Filters - Action filters are filters that are applied when the user clicks on a specific graph representation of the enrolled population. When that segment is selected, the remaining elements on the dashboard will be filtered to represent only that selected segment of the population. Available Action Filter measures are:

Grade

Ethnicity

Gender

IEP Status - Enrolled students who have and IEP or don't have an IEP.

Section 504 Status - Enrolled students who have a Section 504 plan and those who don't.

LEP Status - Enrolled students with Limited Enlish Proficiency and those who are English proficient.

Service Type - The type of enrollment the student receives; designated as either "Primary," "Partial," or "Special Ed Services Only."

State Excluded - Enrollments that are excluded from State of Nevada counts as specified by law.

For example, if "Multiracial" is selected in the "Ethnicity" table,...

...all other tables on the dashboard will be filtered to show only students enrolled who are classified as "Multiracial".

To remove the effect of an Action Filter, click on the selection a second time. Action Filters do not carry forward from one Dashboard to another and are not available on all Dashboards.

Quick Filters - Quick Filters are represented by the dropdown lists on the right side of the dashboard. The available Quick Filters are School Year, Performance Zone, and School Name. Users may select one value for each Quick Filter or they can select "All". Selecting a Quick Filter will narrow the resulting Enrollment to that population only.

Quick Filters applied on one Dashboard will affect all other Dashboards in the workbook.

Parameter Filters - Parameter Filters will break down the analysis by a demographic measure selected using the dropdown list. The available measures are:

Ethnicity

Gender

IEP Status - Enrolled students who have and IEP or don't have an IEP.

Section 504 Status - Enrolled students who have a Section 504 plan and those who don't.

LEP Status - Enrolled students with Limited Enlish Proficiency and those who are English proficient.

Service Type - The type of enrollment the student receives; designated as either "Primary," "Partial," or "Special Ed Services Only."

State Excluded - Enrollments that are excluded from State of Nevada counts as specified by law.

The effect of selecting a Parameter Filter measure in the Current Enrollment Analysis Dashboard is seen on the Grade Demographics table.

Parameter Filters applied on one Dashboard will affect all other Dashboards in the workbook.

Tab Details

Start Tab

The Start Tab contains instructions to use the visualization, contact information, and disclaimers. There is a link to print the manual at the top right above the table of contents.

Current Enrollment Analysis

The Current Enrollment Analysis Dashboard shows the total enrolled population for the current point in time. The enrollment is broken down by:

Grade

Ethnicity

Gender

IEP Status - Enrolled students who have and IEP or don't have an IEP.

Section 504 Status - Enrolled students who have a Section 504 plan and those who don't.

LEP Status - Enrolled students with Limited Enlish Proficiency and those who are English proficient.

Service Type - The type of enrollment the student receives; designated as either "Primary," "Partial," or "Special Ed Services Only."

State Excluded - Enrollments that are excluded from State of Nevada counts as specified by law.

The Grade Demographic can be further broken down using the "Grade By:" Parameter Filter previously discussed.

Weekly Enrollment Analysis

The Weekly Enrollment Analysis Dashboard shows the total enrolled population for every Monday in the current school year. The enrollment can be viewed using the Parameter Filter as previously discussed. The "Total Enrollment" will remain on the Dashboard as the top line.

Enrollment Analysis by Year

The Enrollment Analysis by Year Dashboard shows the total enrolled population using the count day enrollment for every school year to date, beginning with 2007. The enrollment can be viewed using the Parameter Filter as previously discussed. The "Total Enrollment" will remain on the Dashboard as the top line.

Exporting

Should you wish to export this information, you may do so on any visualization by selecting the "Export" tool at the top center of the dashboard. Selecting the "Export" tool will give the following options:

Image

Selecting "Image" will prompt you to save a copy of the screen you have displayed. Select "Download" to save the image.

Data

Selecting "Data" will allow you to export the data displayed on the dashboard to a text file. Note that you must select one bar from one of the graphs in order to activate the Data Export option on each of the tabs. Selecting the Summary tab will allow you to export one record with the displayed information (Example below).

Selecting the Underlying tab will allow you to export the detailed records displayed on the visualization (Example below).

 

Cross tab

Selecting "Cross tab" will allow you to download the desired information directly into an Excel spreadsheet. Selecting "Image" will prompt you to save a copy of the screen you have displayed. Select "Download" to save the exported Excel file (Example below).

 

PDF

Selecting "PDF" will export the desired view to an Adobe Reader, or PDF file. After selecting "PDF", determine your preferences for printing on the screen when prompted. The following options should be determined:

- Portrait or Landscape

- Paper Size

- Scaling: Allows for the selection of the size of the image to be exported.

- Content: Allows for the printing of the selected dashboard, Sheets in Dashboard (selection of specific graphs to be exported), Sheets in Workbook (selection of one or many sheets to be exported).

Once all selections have been made, select the "Export" button to complete the printing / saving of the exported information.
